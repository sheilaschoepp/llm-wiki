# Proxy panel

- **kind**: fictional-roles
- **summary**: A realistic mock-defence proxy committee — exam roles (chair, near-field expert, broader-field examiner, recent passer, optional observer) rather than named people.
- **members**: every profile in `profiles/`. Default questioning roster is chair, near-field expert, broader-field examiner, recent passer; the observer is optional and logistics-only (see its profile).
- **question source**: `../../mock-committee-best-practices.md` — the core question bank (§5), the candidacy-vs-final-defence emphasis (§3), and the session protocol (§4). Each profile fixes a role's lens; the backstop doc seeds the questions.
- **fabrication rule**: these are roles, not real people. Never attribute a real paper, position, or named work to a proxy member. Press from the role's job, the core question bank, and what the document under defence actually says. "Imitate an intellectual perspective, not parody a person" (mock-committee-best-practices §2).
- **use it for**: a format-faithful rehearsal when you want exam-role realism without modelling your specific examiners — or when no real profiles are available.
